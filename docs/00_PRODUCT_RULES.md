# 00_PRODUCT_RULES.md

## Non-Negotiable Product Logic

Partlink is not a classified ads website.

Partlink is an inventory-first marketplace.

Always preserve:

```text
Physical part
→ digital inventory item
→ SKU
→ QR label
→ inventory record
→ marketplace listing
→ buyer search
→ order
→ QR dispatch scan
→ stock update
→ sales record
```

## What matters most

1. Seller can add real stock fast
2. Every part gets SKU and QR
3. Marketplace listings come from inventory
4. Buyer can search and act quickly
5. Seller can dispatch with QR scan
6. Inventory and marketplace stay synced

## MVP first

Prioritize:
- seller verification
- seller mobile dashboard
- manual part upload
- VIN import
- QR generation
- QR print UI
- inventory list
- marketplace
- buyer part page
- buy flow
- seller orders
- admin verification

Do not prioritize:
- advanced analytics
- subscriptions
- AI recommendations
- courier API
- real payment integration
- MyData automation
- native mobile app
- perfect VIN compatibility

## UI language

Primary UI language is Greek.

Tone:
- simple
- professional
- direct
- operational

Avoid:
- SaaS jargon
- playful copy
- English user-facing labels
- overexplaining
